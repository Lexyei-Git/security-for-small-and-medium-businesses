# OSSEC Overview

OSSEC is an open source Host-based Intrusion Detection System (HIDS) that performs log analysis, file integrity monitoring, policy monitoring, rootkit detection, time-based alerting, and active response.

For small and medium businesses, OSSEC provides basic endpoint visibility and alerting for servers and workstations without requiring complex infrastructure.

It is especially useful for:
- Log monitoring across multiple systems
- File integrity checks on critical servers
- Rootkit and malware detection
- Basic compliance monitoring (file changes, policy violations)
- Alerting on suspicious activities

OSSEC uses agent-server architecture, allowing centralized management of multiple endpoints. Note that Wazuh is a popular fork with additional features and active development.
